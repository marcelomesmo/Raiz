void Input::update ( )

Update input entries. Internal use only, don't call this method.

This alg has given me a little trouble.

Axis are only recognized while in motion, knowing that (and to avoid threating SDL_GameControllers outside events), I also placed an interface for the axis movement here.

Check in which direction it is moving.

If the Axis is moving above the positive Dead Zone then it's moving right (1) or else it's moving left (-1).
